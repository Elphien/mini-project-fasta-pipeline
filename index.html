Tentang data

Dataset berisi 94 sekuens nukleotida gen 5.8S rRNA beserta wilayah ITS1 dan ITS2 (Internal Transcribed Spacer) dari berbagai spesies anggrek famili Orchidaceae — genus Cypripedium, Paphiopedilum, Phragmipedium, dan Mexipedium. Setiap sekuens memiliki nomor akses GenBank resmi yang dapat ditelusuri langsung di NCBI Nucleotide (https://www.ncbi.nlm.nih.gov/nuccore/).

Tahapan pipeline

Tujuh langkah berurutan, masing-masing fungsi Python murni tanpa pustaka analisis sekuens eksternal — hanya struktur data dasar: list dan dict.

Membaca file FASTA

02

Inti kode

Dua fungsi yang menjadi jantung pipeline — pembacaan FASTA ke dalam list, dan penghitungan frekuensi nukleotida dengan dictionary. Kode lengkap tersedia di repositori GitHub.

def baca_fasta(filepath): # LIST utama untuk menyimpan seluruh record sekuens daftar_sekuens = [] header, seq_buffer = None, [] with open(filepath, "r") as f: for baris in f: baris = baris.strip() if baris.startswith(">"): if header: daftar_sekuens.append({ "id": header.split()[0][1:], "sekuens": "".join(seq_buffer).upper() }) header, seq_buffer = baris, [] elif baris: seq_buffer.append(baris) return daftar_sekuens def hitung_frekuensi_nukleotida(sekuens): # DICTIONARY untuk frekuensi tiap basa: {'A':.., 'T':.., 'G':.., 'C':..} frekuensi = {} for basa in sekuens: frekuensi[basa] = frekuensi.get(basa, 0) + 1 return frekuensi
